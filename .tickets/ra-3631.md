# Extract shared prompts for cross-tool portability

Extract the review and test coverage prompts into a shared prompts/ directory (prompts/review-prompt.md, prompts/tests-prompt.md). These contain the pure markdown review protocol, checklist, output format, and rules with no tool-specific config. OpenCode agents would reference them via prompt: {file:./prompts/review-prompt.md}. Other tools (Claude Code, Gemini CLI) get thin wrappers that reference the same shared prompts. This keeps review logic in one place and avoids duplicating it per tool.
